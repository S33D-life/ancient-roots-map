# Map Verification

> Ensure trees on the Atlas are accurately placed and properly described.

## Purpose

The Atlas is the heart of S33D. Every tree marker represents a real living being. This skill teaches you how to verify that tree records have accurate coordinates, species identification, and metadata.

## When to Use

- A task asks you to verify tree locations
- You're reviewing imported dataset coordinates
- You notice a tree marker in the wrong place

## Required Inputs

- Access to the S33D Atlas (`/map`)
- Satellite imagery or street-level photos for cross-reference
- The tree record's current coordinates and species data

## Required Outputs

- Confirmation or correction of latitude/longitude (6 decimal places)
- Species verification with source citation
- Photo evidence when available
- Notes on any discrepancies found

## Proof-of-Work Standard

Your submission must include:

1. **Tree ID** — the record you verified
2. **Verification method** — satellite, field visit, cross-reference with external database
3. **Result** — confirmed accurate, corrected (with new values), or flagged for review
4. **Evidence** — screenshot, photo, or link to reference source

## Review Criteria

Curators will check:

- [ ] Coordinates are within 50m of actual tree location
- [ ] Species identification matches visual evidence
- [ ] Source is cited for any corrections
- [ ] No duplicate of existing verified record
