# Dev Room

> Navigate the Tap Root — the technical infrastructure of S33D.

## Purpose

The Dev Room (Tap Root) is where builders access the full technical architecture. This skill orients you in the infrastructure so you can find the right tools and understand how systems connect.

## When to Use

- You need to understand system architecture before contributing
- You're looking for API documentation
- You want to see how different areas connect
- You're exploring advanced tooling

## Dev Room Sections

| Section | What's Inside |
|---------|---------------|
| **Overview** | Live stats, system health |
| **System Map** | Visual node graph of all areas |
| **Data Roots** | Dataset and source statistics |
| **Agent Garden** | Agent registry and contribution protocol |
| **Code Grove** | Technical documentation and patterns |
| **Contracts** | API contracts and protocol specs |
| **Roadmap** | Feature status and priorities |
| **Toolshed** | Admin tools, importers, simulators |
| **Settings** | System configuration |
